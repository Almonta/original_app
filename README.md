# README

## 開発言語
- Ruby2.6.5
- Rails5.2.5

## 就職Termの技術
- devise
- Ajax (コメント機能)
- AWS

## カリキュラム外の技術
- ransack
- Vue.js (優先度:低)

## その他の技術
- お気に入り機能
- kaminari
- bootstrap
- carrierwave
- mini_magick

## 実行手順
```
$ git clone git@github.com:Almonta/original_app.git
$ cd original_app
$ bundle install
$ rails db:create
$ rails db:migrate
$ rails s
```
